# WUT-PMT-tester_complementary_scripts
WUT PMT tester complementary scripts for PMT data analysis
----------------------------------------------------------

These are a collection of PMT analysis scripts, complementary to [**WUT PMT tester**](https://git.hyperk.org/fd2/wut-pmt-tester/wut-pmt-tester), developed by my Hyper-Kamiokande FD2 group colleagues from WUT.


## Scripts

- **extract_pkl.py** – Pulls out data from .pkl files made by [**test_threshold_equalisation.py**](https://git.hyperk.org/fd2/wut-pmt-tester/wut-pmt-tester/-/blob/main/test_threshold_equalisation.py?ref_type=heads) and makes simple table with mean and stdev values for each channel in output file **output_pkl.txt**
